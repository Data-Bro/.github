## CSPSC47200 – Introduction to Data Mining  

### Overview  
This is the official GitHub organization for **Data Bro** for **CSPSC 47200 – Introduction to Data Mining**. 

This organization hosts and manages source code repositories for Data Bro's group assignments, projects, and competitions completed for our Introduction to Data Mining course.

The primary goal of this organization is to provide a centralized location for our course-related project repositories, collaborate, and enforce professional development workflows.
